# TORUS--Custom Porgramming Language
## Two Functional Compiler One Written in Java and the Other: C++
To begin, essentially, this is a solo project I am doing as a highschooler to learn much more about how programming languages actually work. I hit the inevitable tutorial gap, and so I deciding to embrace the rigorous engineering. This is a work in progress and I do not expect to be complete for about a year. I am setting goals on progress to keep this software developing and heading towards the open source world while taking my loaded difficult schedule throughout highschool.
## About The Compiler
  This compiler for my custom language, TORUS, is going to be written mutliple times for efficiency purposes, first in Java. I plan to go down the line of C++, C, and maybe even one day in assembly for a specific computer architecture. Since I am self taught as of this point when it comes to programming, I am using Robert Nystrom's _Crafting Interpreters_ internet guide as a launch pad to learn how the systems that translate source code in order to "send to space" my own project. I plan on heavily optimizing beyond Mr. Nystrom's guide for proof of understanding. 
  Originally, this was supposed to be a project for me and my friend, but so far he has done nothing more than name it. Yes, the name TORUS was not my own invention, rather was the brain child a peer of mine. All infrastructure and programming though has been done by me.
